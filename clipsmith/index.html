Clipsmith

JSON, YAML, Base64, hash tools

50+ clipboard transforms in one menu bar app. Copy a JWT, a JSON blob, a base64 string, a color, a Unix epoch — press one hotkey, pick a transform, paste the result.

Coming to the Mac App Store See features

macOS · Sandboxed · No network · Free

Features

Everything you copy, transformed

Press ⌥⌘V. Clipsmith reads the clipboard, detects what it is, and shows only the relevant tools.

Encode & Decode

Base64, hex, URL components, HTML entities, JSON escape — both directions, instantly.

Hashes

MD5, SHA-1, SHA-256, SHA-512 via CryptoKit. Lowercase hex, ready to paste.

JSON / YAML / XML / CSV

Format, minify, infer TS types, generate Markdown tables, convert in any direction.

JWT decoder

Paste a token, get a pretty-printed header, payload, and signature. Detected automatically.

Numbers & timestamps

Decimal ↔ hex ↔ binary. Unix epoch ↔ ISO 8601. Auto-detected on copy.

Colors

Hex ↔ RGB ↔ HSL. 3-, 6-, and 8-character hex with alpha supported.

Text & lines

snake, kebab, camel, Pascal, CONSTANT, Title, Sentence. Sort, dedupe, trim, count.

Web

Slugify (NFD-aware). Query string ↔ JSON. URL breakdown as Markdown table.

Logs & on-device AI

Strip ANSI, timestamps, k8s prefixes. Apple Foundation Models summarize — no data leaves your Mac.

Privacy

No network. No keystroke monitoring. On-device AI only.

Clipsmith is built so it can't snoop on you, even if you wanted it to. Sandboxed by default; the entitlements simply aren't there.

No network entitlement

Nothing can be uploaded. Period.

No Accessibility permission

No keystroke monitoring. Ever.

On-device AI

Apple Foundation Models, fully local.

History opt-in

Off by default. Auto-redacts secrets.

Per-app exclusion

Exempt 1Password, Bitwarden, banks.

Open-source deps only

GRDB.swift and Yams — both offline.

A look around

The palette adapts to whatever you just copied — JSON, JWT, color, epoch, text.
